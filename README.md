## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* [Expo](https://expo.io/)


### Installation

1. Clone the repo
```sh
git clone git@github.com:willfbren/learn-creative-cloud.git
```
2. Install dependencies
```sh
cd learn-creative-cloud
npm install
```
3. Start Expo server
```sh
expo start
```
